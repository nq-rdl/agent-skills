## 0.3.0 - 2026-04-16
### Added
* New `argo-cd` skill — ArgoCD CLI usage, declarative config, Helm/Kustomize/OCI support, and project auditing (#42)
* New `conventional-commits` skill — teaches agents the Conventional Commits 1.0.0 spec for structured commit messages (#15)
* New `stitch-design-md` skill for generating Google Stitch UI specs — includes RST format overview and examples (#49)
* New `sops` skill for encrypting and decrypting files — supports Age and HashiCorp Vault backends (#17)
* New `/agent-teams` slash command for `cc-agent-teams` skill — designs agent teams interactively with role, sizing, and ownership guidance
### Changed
* Adds RST reference files for all skills and Speckit integration scaffolding for Claude, Gemini, and OpenCode (#45)
### Fixed
* Fixes `gemini-cli` skill using invalid `--approval-mode yolo` flag — replaced with `--yolo` across all examples (#40)
